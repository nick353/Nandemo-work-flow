# HEARTBEAT.md

## 📋 定期チェックリスト

### 自動バックアップ（2時間ごと）

現在時刻を確認して、前回のバックアップから2時間以上経過していたら：

1. 以下のコマンドを実行してバックアップ:
   ```bash
   cd /root/clawd && GITHUB_TOKEN=${GITHUB_TOKEN} ./auto-backup.sh
   ```
   ※ GITHUB_TOKENは環境変数から自動的に読み込まれます

2. 完了したら「✅ バックアップ完了」と報告

3. 最終バックアップ時刻を `backup-state.json` に記録

### バックアップ状態の確認

- 最終バックアップファイルの確認: `/root/clawd/backup-state.json`
- もしファイルが存在しない、または2時間以上前なら → バックアップ実行
- 最近バックアップ済みなら → スキップ

### その他のチェック（任意）

- メモリーファイルの更新確認
- 重要な変更があれば即座にバックアップ
