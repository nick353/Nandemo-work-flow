## 失敗記録

### [2026-02-09 01:57] バックグラウンドタスクの完了報告忘れ

**何が起きた：**
- 01:44に pnpm install をバックグラウンド実行
- 「完了したら報告する」と約束
- でも報告しなかった（忘れた）
- andoさんが「先ほどのダウンロードはどう？」と質問
- 混乱して正確な状況を把握できなかった

**原因：**
1. バックグラウンドタスク完了時に自動通知がない
2. 次の会話まで忘れてしまう
3. 手動確認しないと気づけない
4. RUNNING_TASKS.md と実際の状態が同期していなかった

**実装した対策：**
1. **HEARTBEAT.md更新** - タスクチェックを最優先に追加
2. **SOUL.md更新** - より厳格なタスク管理ルールを追加
3. **RUNNING_TASKS.md更新** - 詳細テンプレート追加
4. **このファイル作成** - 失敗を記録・学習する仕組み

**今後の行動：**
- タスク開始時：必ずRUNNING_TASKS.mdに詳細記録
- 質問時：必ず `process list` で確認
- ハートビート時：自動チェック＆報告
- 約束は必ず守る

## 学習ルール

1. **失敗したらすぐ記録** - このファイルに追加
2. **原因を深掘り** - なぜ起きたか？どう防ぐか？
3. **対策を実装** - 即座に改善策を適用
4. **定期レビュー** - このファイルを読み返して徹底
